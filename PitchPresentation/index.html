How to use it

  • choose a training method, currently a choice of Linear Regression, Generalized Linear Model, and Polynomial Kernel

  • A model gets trained on available data to predict eruption duration

fit <- train( eruptions ~ waiting, data = train, method = "lm")
  • The out of sample error is reported on the test data
sqrt( sum( (predict( fit, newdata = test ) - test$eruptions)^2 ) )
